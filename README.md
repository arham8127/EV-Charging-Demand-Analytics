# ⚡ EV Charging Demand Analytics

A data analytics project that explores electric vehicle (EV) adoption trends and charging station infrastructure using Python. The project analyzes EV growth, vehicle types, electric range distribution, and charging station data to generate useful insights and visualizations.

---

##  Project Objective

The objective of this project is to analyze the rapid growth of electric vehicles and understand the demand for EV charging infrastructure. By analyzing EV population data and charging station datasets, the project identifies trends, distributions, and key insights related to EV adoption.

---

##  Datasets Used

1. **EV Population Dataset**
   - Contains data about electric vehicle registrations
   - Includes model year, manufacturer, EV type, and electric range

2. **Charging Stations Dataset**
   - Contains information about EV charging stations
   - Includes charger type, charging capacity, operator, and usage statistics

##  Key Insights

- EV adoption has grown significantly over recent years.
- Battery Electric Vehicles (BEVs) dominate the EV market.
- Tesla is one of the most popular EV manufacturers.
- Fast chargers are the most commonly used charging infrastructure.
- EV charging infrastructure is expanding globally to meet rising demand.
